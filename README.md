# 🇵🇰 Pakistan Fuel Price Analysis & Forecasting

An end-to-end Data Science and Machine Learning project that analyzes historical fuel prices in Pakistan (2023-2026) and forecasts future prices using Linear Regression.

## 🚀 Features
- **Data Visualization**: Detailed trend analysis using `matplotlib` and `seaborn`.
- **Machine Learning**: Future price prediction using `scikit-learn` (Linear Regression).

## 🛠️ Tech Stack
- Python 3
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-Learn
